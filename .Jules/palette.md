# Palette's Journal

## 2025-05-15 - Sidebar Accessibility
**Learning:** In a neo-brutalist design with icon-heavy sidebars, ensuring screen readers can distinguish the active page and identify icon-only buttons (common in mobile views) is critical for inclusive navigation.
**Action:** Use `aria-current="page"` for active links and `aria-label` for all navigation items to support both desktop and mobile users.
